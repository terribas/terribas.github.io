Random Walker - Introduction


This project has been created by Alejandro Terribas Ruiz for the Physics course corresponding to the Computer Science degree at Universitatea Tehnica din Cluj-Napoca.

The chosen topic is the implementation of a random walker using JavaScript along with the p5.js library.

To give utility and meaning to the random walker, this project includes, among other things, the explanation of the importance of random numbers in computer science, how they are generated, and the properties a pseudo-random number generator (PRNG) must satisfy. These properties can be visually verified using the graphical visualizers created with p5.js for this project.
