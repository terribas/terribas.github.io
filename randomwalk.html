Random Walker - First Approach

This coding challenge suggested for this subject gives us a very good first approach of a random walker. This allows us to check the efectiveness of our pseudo-random numbers generator. In this case, our random numbers come from the built-in javascript pseudo-random numbers generator: 
            
// Daniel Shiffman
// http://codingtra.in
// https://youtu.be/l__fEY1xanY
// https://thecodingtrain.com/CodingChallenges/052-random-walk.html

let x;
let y;

function setup() {
   createCanvas(400, 400);
   x = width / 2;
   y = height / 2;
   background(51);
}

function draw() {
   stroke(255, 100);
   strokeWeight(2);
   point(x, y);
   const r = floor(random(4));
   switch (r) {
      case 0:
      x = x + 1;
      break;
      case 1:
      x = x - 1;
      break;
      case 2:
      y = y + 1;
      break;
      case 3:
      y = y - 1;
      break;
   }
}
However, we can always improve it to make it look better. Some improvements could be adding a button to start the random walk, as well as adding the possibility to change the speed and randomly change the color of the path. We can see an improved version here.
